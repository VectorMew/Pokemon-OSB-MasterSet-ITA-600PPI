# 🎒 Pokémon TCG - OBF Master Set [Italiano] (600 PPI / 300 PPI)

Benvenuti in questo archivio open-source dedicato alla riproduzione e al playtest del set OBF con testi interamente in italiano. Questo progetto nasce per permettere ad appassionati e collezionisti di stampare copie di altissima qualità per uso personale, salvaguardando le carte originali da usura e graffi.

---

## 📂 Contenuto del Progetto

All'interno di questo repository troverai due modalità di utilizzo:

1. **`FULL_MasterSet_OBF_ITA.pdf` (600 PPI):** Un unico file PDF da 115 MB pronto per la stampa. Contiene 52 pagine totali disposte in sequenza alternata (Pagina 1: Fronte, Pagina 2: Retro, Pagina 3: Fronte, Pagina 4: Retro...) ottimizzato per il formato A4.
2. **`TCG_FullSET_OBF_ITA_PNG.zip` (300 PPI):** Un archivio compresso contenente tutte le 230 carte singole del set in formato PNG ad alta definizione, ideali se vuoi stampare solo carte specifiche o per l'utilizzo su simulatori digitali.

---

## 🖨️ Guida alla Stampa Professionale (Errore Zero)

Per fare in modo che le carte vengano stampate alle dimensioni reali al millimetro e con i colori corretti (convertiti nativamente in spazio colore **CMYK**), imposta il software di stampa così:

* **Software:** Apri il file con **Adobe Acrobat Reader** o **Photopea** (non stampare direttamente dal browser internet).
* **Adattamento di Stampa:** Seleziona tassativamente **"Dimensioni Reali"** o **"Senza Modifiche"** (NON usare "Adatta all'area di stampa", altrimenti le carte verranno rimpicciolite).
* **Orientamento & Rotazione:** Imposta il foglio in **Verticale** e seleziona la rotazione sul **Lato Lungo** (stile libro, da destra verso sinistra). Questo riduce l'errore meccanico di disallineamento fronte/retro a meno di 1 mm.
* **Supporto consigliato:** Cartoncino da **300g** (finitura lucida o patinata).

## 🎨 Consigli per il Fai-Da-Te (Finitura)
* **Taglio:** Per evitare sbavature, effettua i tagli lineari tenendo il foglio dal lato del Fronte.
* **Angoli:** Utilizza una fustella per angoli (Corner Rounder) impostata sul raggio **R4** o **R5** per ottenere la curvatura perfetta dei collezionabili.
* **Effetto Olografico:** Applica una pellicola adesiva *Rainbow* sottile prima di tagliare il foglio per simulare l'effetto "Holo Rare" delle carte originali.
